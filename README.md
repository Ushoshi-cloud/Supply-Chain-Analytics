# Supply-Chain-Analytics
Power BI dashboard showcasing key supply chain metrics, including revenue, costs, stock levels, transportation modes, supplier performance, and city-wise supply volume—built using SKU-level data to deliver actionable operational insights.
Supply Chain Performance Dashboard – Power BI

This project presents a Power BI Supply Chain Performance Dashboard built using a structured dataset containing SKU-level information, supplier data, transportation modes, shipping metrics, and financial performance indicators. The dashboard provides a unified view of revenue, costs, stock availability, logistics efficiency, and supplier contribution to support data-driven decision-making.

Project Overview
This project analyzes the end-to-end supply chain by integrating product data, sales metrics, inventory levels, and transportation details.
The dashboard highlights key KPIs such as:

Total Revenue
Total Cost
Total Orders
Total Stock Level
Category-wise Revenue
City-wise Supply Volume
Supplier Contribution
Transportation Mode Share
Shipping Carrier Performance

It helps identify trends, bottlenecks, and opportunities for improving supply chain efficiency.
